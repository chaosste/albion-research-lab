# Local path conventions

Canonical local checkout for Albion Research Lab:

- `/Users/stephenbeale/Projects/albion-research-lab`

Companion OMX source checkout (optional; this repo vendors it as a submodule):

- `/Users/stephenbeale/Projects/oh-my-codex`

Guidance:

- Use the albion-research-lab path in new notes, scripts, and policy files.
- Run `./omx` from the repo root; it resolves to the pinned `oh-my-codex` submodule.
